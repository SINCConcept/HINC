we install it via yum: http://fiware-orion.readthedocs.org/en/develop/admin/install/index.html#using-yum-(recommended)

requirement: Centos 6 (not Centos7), MongoDB.
Other dependencies are taken care by yum

May need to install these:
sudo yum install python-setuptools python-setuptools-devel
sudo easy_install pip
pip install flask

then to run accumulate server:
# cd /dir/where/accumulator-server/is/downloaded
# chmod a+x accumulator-server.py
# ./accumulator-server.py 1028 /accumulate ::1 on


Start mongod: service mongod start
Start contextBroker service: service contextBroker start/restart


Drop database:
$ mongo localhost/orion
> db.dropDatabase()
> exit
