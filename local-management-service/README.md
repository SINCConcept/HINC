# HINC local service

## Overview
This package contain core functions of HINC local service.
A local service will integrate various providers (IoT, Networks and Clouds). 

## Running Local
java -jar local-management-service-1.0.jar

## Configuration
You will need to provide an application.properties file in the same directory. A sample can be found in src/main/resources 

## Limitation
- Secure HTTPS
